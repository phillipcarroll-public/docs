# XPU Ollama Setup Windows

Github Repo: <a href="https://github.com/intel/ipex-llm/blob/main/docs/mddocs/Quickstart/ollama_portable_zip_quickstart.md">https://github.com/intel/ipex-llm/blob/main/docs/mddocs/Quickstart/ollama_portable_zip_quickstart.md</a>

Download zip: <a href="https://github.com/ipex-llm/ipex-llm/releases/tag/v2.3.0-nightly">https://github.com/ipex-llm/ipex-llm/releases/tag/v2.3.0-nightly</a>

Unzip and run: start-ollama.bat

This will open and start the service in a new window. From there you can pull ollama models, note that this version does NOT support MXFP4 so some newer models may not run.
